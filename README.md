# amazon-product-review-sentiment-analysis
Sentiment Analysis on Amazon Product Reviews using NLP and Machine Learning. Includes data preprocessing, exploratory analysis, TF-IDF vectorization, model training, evaluation, and prediction script.

This project focuses on performing Sentiment Analysis on Amazon Product Reviews using Natural Language Processing (NLP) techniques. The goal is to understand customer opinions by classifying reviews into Positive or Negative sentiments.
The workflow includes data cleaning, text preprocessing, TF-IDF vectorization, model training, and evaluation using machine learning algorithms.
This repository serves as a complete end-to-end implementation, from raw text data to a fully trained sentiment classifier.
